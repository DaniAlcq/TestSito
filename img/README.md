# Cartella foto

Organizza le foto in sottocartelle, poi nell'admin scrivi il percorso relativo (es. `img/news/foto.jpg`).

## Struttura consigliata

```
img/
├── logo.svg                 ← già presente
├── news/                    ← foto degli articoli news
│   ├── 2025-11-09-vittoria.jpg
│   └── ...
├── squadre/                 ← foto di gruppo squadre
│   ├── serie-c-f.jpg
│   ├── serie-c-m.jpg
│   └── ...
├── atleti/                  ← foto singole atlete/atleti/staff
│   ├── russo-anna.jpg
│   ├── bianchi-luca.jpg
│   └── ...
└── sponsor/                 ← loghi sponsor
    ├── fima.png
    ├── adg.png
    └── ...
```

## Dimensioni consigliate (per velocità!)

| Tipo | Dimensione | Note |
|---|---|---|
| Foto squadra | 1200×900 px | jpg, qualità ~80% |
| Foto news | 1200×750 px | jpg, qualità ~80% |
| Foto atleta | 600×800 px | jpg, qualità ~80%, verticale 3:4 |
| Logo sponsor | PNG trasparente, alto 200 px | png |

## Suggerimento

**Prima di caricare:** comprimi le foto con https://squoosh.app (gratis, online, niente upload).
Una foto da telefono pesa 4MB, dopo compressione 150KB → sito molto più veloce.
